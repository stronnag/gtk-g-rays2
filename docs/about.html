About gtk-g-rays2

gtk-g-ray2 was written by Jonathan Hudson. To find more information about gtk-g-ray2, please visit the gtk-g-ray2 Web page.

An initial Maemo port was contributed by Tilman Vogel. gtk-g-rays2 v1.x after 1.06 builds for desktop Linux (GTK+-2) and Maemo, gtk-g-rays2 2.x is desktop Linux (GTK+-3) only.

To report a bug or make a suggestion regarding this application or this manual, please contact the author.
